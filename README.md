# ⚡ JS Mini Projects

Coleção de mini projetos em JavaScript focados em lógica de programação e resolução de problemas do mundo real.

---

## 📂 Projetos

### 🌬️ Índice de Qualidade do Ar — Florianópolis

`air_quality_floripa_index.js`

Avalia o índice de qualidade do ar com base em parâmetros de entrada e retorna a classificação correspondente.

**Conceitos aplicados:** estruturas condicionais, intervalos numéricos, classificação de dados

---

### ⚖️ Calculadora de IMC

`imc_calculator.js`

Calcula o Índice de Massa Corporal e retorna a faixa de classificação (abaixo do peso, normal, sobrepeso, etc).

**Conceitos aplicados:** operações matemáticas, condicionais encadeadas, validação de entrada

---

### 🪙 Cara ou Coroa

`coin_flip_index.js`

Simulação de lançamento de moeda com resultado aleatório.

**Conceitos aplicados:** geração de números aleatórios, `Math.random()`, lógica booleana

---

## 🛠️ Tecnologias

- JavaScript (ES6+)
- Node.js

---

## ▶️ Como executar

```bash
# Clone o repositório
git clone https://github.com/debora-arruda/js-mini-projects.git

# Execute qualquer projeto
node air_quality_floripa_index.js
node imc_calculator.js
node coin_flip_index.js
```
